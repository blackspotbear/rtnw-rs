## What's this?

rtnw-rs is the next week ray tracer in Rust, which is following [@Peter_shirley](https://twitter.com/Peter_shirley)'s textbook.

## Usage

To get an image run:

```sh
cargo run --release
```

To show help run:

```sh
cargo run --release -- --help
```
